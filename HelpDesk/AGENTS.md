# AGENTS.md

## 프로젝트 개요
Google ADK 학습을 위한 미니 프로젝트입니다.
사내 IT 헬프 데스크 에이전트를 완성하는 것이 목표입니다.

## MVP
1. 사용자 질문의 카테고리를 분류합니다.
2. FAQ 검색 tool을 사용하여 답변합니다.
3. 답변 불가 시 티켓(로컬 TXT 파일) 생성 tool을 호출합니다.
4. 최종 응답을 정해진 형식으로 출력합니다.

## 기술 스택
- FastAPI를 기반으로 사용합니다.
- 에이전트는 Google ADK 프레임워크로 구현합니다.
- FAQ 데이터는 MongoDB에 저장합니다.
  - 필요한 경우 벡터 서치를 활용합니다.
- Arize Phoenix로 에이전트 답변을 모니터링합니다.

## 아키텍처
- api/ 하위에 FastAPI 코드를 작성합니다.
- helper/ 하위에 에이전트 코드를 작성합니다.
- mongodb/ 하위에 DB 관련 파일을 작성합니다.
- phoenix/ 하위에 모니터링 관련 파일을 작성합니다.

## 규칙
- 별도 요청이 있을 때만 코드를 생성 및 수정합니다.
- 승인 없이 의존성을 생성하지 않습니다.
